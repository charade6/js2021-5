### 자동  자료형 변환
1. **숫자와 문자열 자료형 자동변환**
숫자와 문자열에 '+' 연산자 적용하면 자동으로 숫자가 문자열로 변환
ex)`console.log("52"+273);`
52273   // 출력
2. **불 자료형 자동변환**
'!'연산자를 두 번 사용해 Boolean()함수 사용하는것과 같은 효과
ex)`console.log(!!0);`

### 식별자
